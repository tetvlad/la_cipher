# Шифр Цезаря

Java‑приложение для **кодирования и декодирования текстовых файлов** методом шифра Цезаря, с возможностью
задавать сдвиг.
Поддерживает **только русский алфавит (33 буквы)** и работу с внешними файлами в кодировке UTF‑8.

###### USE Case 1: Кодирование файла

Шифрование `file1.txt` → `file2.txt`  
(Enter = shift от 1 до 32)

###### USE Case 2: Декодирование файла

Дешифрование `file2.txt` → `file3.txt`  
(Enter = shift от 1 до 32)

###### 1) Core: Декодирование файла

CaesarApp.java          ← Меню + ввод сдвига, по умолчанию сдвиг равен 3  
CaesarProcessor.java    ← Обработка файлов  
CaesarCoder.java        ← Шифрование/дешифрование  
CaesarAlphabet.java     ← Русский алфавит

###### 2 Model (Domain) - модель данных

###### 3) Service - работа с файлами

FileService.java        ← Чтение/запись файлов UTF-8  
ValidationService.java  ← Проверка символов/файлов


###### 4) Exeption - кастомные исключения

CaesarException.java  
CaesarRunTimeException.java

###### 5) Ресурсы - настройка

Сдвиг по умолчанию: 3 (можно задавать до 32)  
Алфавит: 33 буквы
